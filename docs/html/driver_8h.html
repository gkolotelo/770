Detailed Description

File containing the definition of methods for the operation of an H-Bridge in bipolar operation mode. File containing the definition of methods for the operation of an H-Bridge in bipolar operation mode.

Author
Guilherme Kairalla Kolotelo
Bruno de Souza Ferreira
Version
1.1
Date
10 Jun 2016
27 Sep 2016
  • Driver actuation through driver_setDriver goes from -100 to 100, the former being full reverse, and the latter, full steam ahead.
  • Driver max freq @ 100kHz.
  • Channel A on HighTrue and Channel B on LowTrue makes for a bipolar H-Bridge pair of control signals.
